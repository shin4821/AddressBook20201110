# 주소록
주소록 만들기 프로젝트

####기간: 2020.10.14 ~ 2020.11.10
####사용언어: C
####사용기술: 직접 Array Library를 만들어서 사용

프로젝트 상세내용
=============
목적
-------------
<pre>
<code>
* 주변의 지인들의 연락처 관리용
</code>
</pre>

개발인원
-------------
<pre>
<code>
* 본인 한명
</code>
</pre>

주요기능
-------------
<pre>
<code>
1. 사람들의 연락처를 기재한다.
2. 이름을 검색하여 찾다.
3. 연락처가 변경되었을 경우 고친다.
4. 필요가 없는 연락처는 지운다.
5. 사람들의 이름을 기준으로 정렬한다.
</code>
</pre>
